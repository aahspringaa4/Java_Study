# 내용 정리, 헬로 월드!

### 그동안 배운 내용 정리

#### 클래스란 자바 프로그램의 최소 단위이다.

```
public class foo{

}
```

#### 주석은 실행 되지 않는 코드이다.

```
// 라인 주석 
/*
블록 주석
*/
```

#### 메인 메소드란 프로그램 실행의 시작점이다.

```
public static void main(String[] args){
}
```

#### println()메소드는 문자열을 출력한다.

```
public class Main {

    public static void main(String[] args) {
        System.out.println("Hello My name is yoo");
    }
}
```

#### Hello World!

```
public class Main {

    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```
