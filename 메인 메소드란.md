# 메인 메소드란

**메인 메소드란**, 프로그래밍 실행의 시작점입니다. **메인 메소드**는 클레스 내부에 위치하며 아래와 같은 구조를 갖습니다.

```
public class Pizza{
    public static void main(String[] args){
    }
}
```

## println()메소드란

println()메소드는, 소괄호 안의 문자열을 출력합니다.

```
public class Main {

    public static void main(String[] args) {
        System.out.println("저는 유현명입니다.\n열심히 하겠습니다.");
    }
}
```

#### 출력

저는 유현명입니다.

열심히 하겠습니다.
