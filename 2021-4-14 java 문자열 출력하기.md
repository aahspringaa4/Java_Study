# java 문자열 출력하기



```
public class Hello {

     public static void main(String[] args){

        System.out.println("Hello World!);

     }

}


```



***\*Hello World!\****가 출력이 된다.



## 응용해보기!



```
public class Hello {

   public static void main(String[] args){

      System.out.println("Hello World\what a beautiful day");

   }

}
```



***\*Hello World\****

***\*what a beautiful day\****가 출력이 된다.
